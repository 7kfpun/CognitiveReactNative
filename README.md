## Running

#### Clone & install

* Clone this repo `git clone git@github.com:7kfpun/CognitiveReactNative.git`
* `cd CognitiveReactNative`
* run `npm install`

#### iOS

* Run `react-native run-ios`

#### Android

* Run `android avd` and start an emulator
* Run `react-native run-android`
